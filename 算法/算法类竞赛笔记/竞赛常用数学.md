# 最大公约数

使用==*辗转相除法*==，又称*欧几里德算法（Euclidean Algorithm）*，是求两个数的*最大公约数（greatest common divisor）*的一种方法。**用较大的数除以较小的数，再以除数和余数反复做除法运算，当余数为0时，取当前算式除数**为最大公约数。

例如：求30和18的最大公约数：

​			30/18=1余12

​			18/12=1余6

​			12/6=2余0

​			30和18的最大公约数为6
