# 归并排序

归并排序的思想是先递归左右两边，在双指针扫描排序合并

```c++
void merge_sort(int q[], int l, int r)
{
    if (l >= r) return;//边界

    int mid = l + r >> 1;//每次折半
    递归左右
    merge_sort(q, l, mid);
    merge_sort(q, mid + 1, r);

    int k = 0, i = l, j = mid + 1;
    //k控制合并后的数组//i控制原数组左边//j控制原数组右边
    while (i <= mid && j <= r)
        if (q[i] <= q[j]) tmp[k ++ ] = q[i ++ ];
        else tmp[k ++ ] = q[j ++ ];
	//把剩余的元素存进去
    while (i <= mid) tmp[k ++ ] = q[i ++ ];
    while (j <= r) tmp[k ++ ] = q[j ++ ];
	//复制到原数组中
    for (i = l, j = 0; i <= r; i ++, j ++ ) q[i] = tmp[j];
}
```
